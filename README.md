# Dana Static
Static website for dana.id.
This static website build using React 16 with `create-react-app` boilerplate.

### [LIVE PREVIEW](https://ariksasmita.github.io)

## Key Features
* Non-relative imports with help from .env NODE_PATH
* Hero quotes, gallery data and about us data are fetched from API and stored to redux storage.
* PropTypes type checking
* Jest tests
* Formik form validation on Contact page

## What can be improved
* Fetched images can be cached using sessionStorage
* More thorough tests for components as well as other functions (redux related files)
